Instituto Rio Maranguapinho - Site Gerenciável
Bem-vindo ao repositório do site do Instituto Rio Maranguapinho! Este projeto foi desenvolvido com o objetivo principal de oferecer uma plataforma totalmente gerenciável para os administradores da instituição. Os esforços conjuntos de Eneas Nobre, responsável pelo backend, e Maxwell Venício, encarregado do frontend, resultaram em uma aplicação dinâmica e funcional.

Funcionalidades Principais
Gerenciamento Total: Os administradores podem facilmente controlar e atualizar o conteúdo do site, mantendo-o sempre atualizado e relevante.

Interface Intuitiva: O frontend, projetado por Maxwell Venício, oferece uma interface amigável e atrativa para os usuários, facilitando a navegação e a compreensão do conteúdo.
